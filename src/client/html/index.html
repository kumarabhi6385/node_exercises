Interview Assesment

All questions are categories under topic related to specific category and subcategory!

Category

It points to main technologies.

Like .NET ecosystem, Node, Azure and AWS

SubCategory

It will belong to any specific technology(here marked under category)

For e.g. express under node, ASP.NET core and ASP.NET under .NET etc.

Topics

Topics will belongs to specific category / subcategory

For e.g. Session Management in ASP.NET, Security in express etc.
